Logistic Regression

Navigating Probabilities, Predicting Outcomes

Logistic Regression is a statistical method used for binary and multiclass classification. Despite its name, it is a classification algorithm rather than a regression one. It's a fundamental algorithm in machine learning, particularly in the field of supervised learning. It is a data analysis technique that uses mathematics to find the relationships between two data factors. It then uses this relationship to predict the value of one of those factors based on the other.

Objective

  • The primary objective of logistic regression is to model the probability of an instance belonging to a particular class.
  • It predicts the probability that a given input belongs to a certain category by fitting a logistic curve to the data.

Results and Insights

Upon conducting Principal Component Analysis (PCA) on the dataset, it was observed that the first 12 features collectively explain approximately 97.25% of the variance. Notably, after dropping the last feature, a marginal decrease in accuracy from 0.8218 to 0.8213 was observed.

Further investigation revealed that training the model with only the first 12 features resulted in an accuracy boost, reaching 0.8227. This indicates that these features contribute optimally to the predictive performance.

In contrast, combining the last three features alone explained around 11.83% of the variance, but excluding them led to a notable decrease in accuracy to 0.8187.

Conclusion

  • In conclusion, the primary goal of maximizing accuracy in this analysis was successfully achieved by leveraging the first 12 features, resulting in an impressive accuracy of 0.8227. This highlights the pivotal role of thoughtful feature selection in influencing model performance.
